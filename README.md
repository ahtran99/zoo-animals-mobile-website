  <h1>Zoo Animals Mobile Website</h1>

  <h2>Description</h2>
  <p>
    A mobile-first web application displaying a list of zoo animals, allowing users to toggle between different animal classifications.
    The app stores data in local storage for offline access and dynamically updates the UI using JavaScript and jQuery.
  </p>

  <h2>Features</h2>
  <ul>
    <li>Displays a list of animals categorized by type (e.g., Mammals, Birds, Reptiles)</li>
    <li>Uses jQuery for DOM manipulation to display animal data</li>
    <li>Stores selected animal data and classifications in Local Storage for persistent access</li>
    <li>Toggle visibility of descriptions for each animal</li>
    <li>Mobile-first design with responsive layout</li>
  </ul>

  <h2>Tech Stack</h2>
  <ul>
    <li><strong>HTML5</strong> &ndash; Structure of the webpage</li>
    <li><strong>CSS3</strong> &ndash; Styling and layout (Flexbox for responsiveness)</li>
    <li><strong>JavaScript</strong> &ndash; For DOM manipulation, data handling, and local storage</li>
    <li><strong>jQuery</strong> &ndash; For smooth UI interactions (show/hide toggle, data loading)</li>
  </ul>

  <h2>What I Learned</h2>
  <ul>
    <li>How to use JavaScript and jQuery to manipulate the DOM dynamically</li>
    <li>Storing and retrieving data in Local Storage for persistent web apps</li>
    <li>Creating a mobile-responsive layout using CSS Flexbox</li>
    <li>Building an interactive UI with minimal dependencies</li>
  </ul>
